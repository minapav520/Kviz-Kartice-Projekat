# Interaktivne Kviz Kartice 🧠

Ovaj projekat je web aplikacija kreirana za lakše učenje putem digitalnih kviz kartica. 
Projekat je rađen u okviru predmeta **Elektronsko poslovanje** na Akademiji strukovnih studija Šumadija.

## 🚀 Tehnologije
* **HTML5** - Struktura stranice
* **CSS3 & Bootstrap 5** - Stil 
* **JavaScript** - Logika za prikaz i promenu kartica

## ✨ Funkcionalnosti
* Prikaz pitanja i odgovora na klik (flip efekat).
* Jednostavna navigacija kroz set kartica.
